# PWA Icons

Place your app icons here in the following sizes:

- `icon-72x72.png` - Small icon
- `icon-96x96.png` - Medium icon
- `icon-128x128.png` - Medium-large icon
- `icon-144x144.png` - Tile icon
- `icon-152x152.png` - Apple touch icon
- `icon-192x192.png` - Standard PWA icon (required)
- `icon-384x384.png` - Large icon
- `icon-512x512.png` - Splash screen icon (required)

## Optional Screenshots

For better PWA experience, also add:

- `screenshot-wide.png` - 1280x720 landscape screenshot
- `screenshot-narrow.png` - 750x1334 portrait screenshot

## Generating Icons

You can use tools like:
1. [PWA Asset Generator](https://www.pwabuilder.com/imageGenerator)
2. [Real Favicon Generator](https://realfavicongenerator.net/)
3. [Maskable.app](https://maskable.app/editor) for maskable icons

## Icon Guidelines

- All icons should be PNG format
- Use your logo/brand on a solid background color (#0ea5e9 recommended)
- For maskable icons, keep important content in the center 80% safe zone
